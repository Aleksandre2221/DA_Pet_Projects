## Description
This project analyzes bike sales data using Power BI, leveraging Power Query, DAX, and interactive dashboards.
The goal of the project is to explore sales performance, product trends, customer behavior, and employee contribution across all stores.

The Power BI report consists of two pages:
- Overview â€” overall sales, product, and store performance
- Staff Performance â€” detailed employee-level sales insights

The dashboard provides insights such as:
- Total revenue and profit analysis
- Sales by store, category, and brand
- Top-performing and low-performing products
- Monthly performance trends
- Discount impact on revenue
- Staff sales comparison and ranking

## Key Features
- Automated data loading & transformation using Power Query
- Clean star-schema data model
- Custom KPIs calculated with DAX
- Interactive slicers for store, brand, category, date, and staff
- Revenue, margin, discount, and product insights
- Employee sales analysis across all stores
- Fully interactive Power BI dashboard
