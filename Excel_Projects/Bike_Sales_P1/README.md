## Description
This project analyzes bike sales data using **Excel**, leveraging **Power Query**, **Power Pivot**, **Pivot Tables**, **DAX**, and interactive dashboards.  
The goal of the project is to explore sales performance, customer behavior, product profitability, inventory levels, and store performance.

The dashboard provides insights such as:
- Total revenue and profit analysis  
- Sales by store, category, and brand  
- Discount impact on sales  
- Customer demographics  
- Product-level analysis  
- Inventory vs. sales performance

## Key Features
- Automated data loading & cleaning using **Power Query**
- Star-schema data model using **Power Pivot**
- Custom KPIs calculated with **DAX**
- Filtering by store, category, brand, date and product
- Revenue, margin, discount, and inventory insights
- Fully interactive Excel dashboard
